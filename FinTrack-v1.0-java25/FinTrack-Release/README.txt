REQUIREMENTS
------------
• Java 17 or later (optimised for Java 25)
  Download free: https://adoptium.net

HOW TO RUN
----------
Windows:   double-click  run-windows.bat
           OR in Command Prompt: java -jar finance-tracker.jar

Mac/Linux: ./run.sh
           OR in Terminal:       java -jar finance-tracker.jar

DATA STORAGE  (100% offline, no internet needed)
------------
  Windows:   C:\Users\<YourName>\FinanceTrackerData\
  Mac/Linux: ~/FinanceTrackerData/

  users.json         — accounts
  transactions.json  — all records

Back up by copying that folder anytime.

FEATURES
--------
  ✔ Register / Login with animated loading screens
  ✔ Dashboard: income, expenses, balance, budget cards
  ✔ 6-month income vs expenses bar chart
  ✔ Top spending category breakdown
  ✔ Add transactions (14 expense + 8 income categories)
  ✔ Filter by type, month, or search keyword
  ✔ Export to CSV / Import from CSV
  ✔ Monthly budget tracker with colour-coded progress bar
  ✔ Edit profile & settings
  ✔ Fully offline — zero network calls

SHARING
-------
Zip this entire folder and send it to anyone.
Each person's data stays on their own machine.

Built with: Java 25 · JavaFX 11 · Apache Commons CSV
